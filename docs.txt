HTTP GET /api/files/:path
  gets file at this path

HTTP POST /api/files/
  - parent path
  - file metadata
  - raw file data
  uploads file

HTTP DELETE /api/files/
  - path
  deletes file at this path

HTTP PUT /api/files/
  - path
  - file metadata
  updates file or file metadata i.e. sharing

File Tree:
{
  username: String,
  uuid: String,
  files: [{
    name: String, // file name
    path: String, // full file path
    isDir: Boolean, // true if dir
    fileType: String, // file extension type
    lastModified: Date, // last modified date
    files: [{path: String}], // file paths of descendent files (if dir)
  }],
  public: [{path: String}]
}
